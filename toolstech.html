Tools and Technology

Unity version 2021.2.12

Unity is a C# engine that runs on any computer running on 64 bit version of Windows 7 SP1+, 8 and 10, Mac OS X 10.12 and above or Ubuntu 16.04, 18.04 and CentOS 7. It also requires a graphics card with DirectX 10 support. The price of Unity is dependent on revenue or funding of creators. It is free as a student or with revenue of less than $100,000 in the previous 12 months. Above this range, Unity requires a subscription. Between $100,000 and $200,000 revenue, it costs $399 a year per seat for the Plus subscription. If revenue is above $200,000, it costs $1800 a year per seat for the Pro subscription, but there is also the Enterprise subscription that costs $4000 a month for 20 seats.
Vision Smiths has no prior experience in using Unity or coding in C#.
https://unity3d.com/get-unity/download https://store.unity.com/compare-plans?currency=USD

Github Version 2.9.9

Github is a free application that runs on Windows 7 64 bit and later, macOS 10.12 and later, Android 6.0 and later, Apple devices with iOS 14.0 and later, or can simply be accessed from the browser. It’s free to use but offers subscription services for more features. The Team subscription is $48 per user for a year and the Enterprise subscription is $252 per user per year.

Figma

Figma is a graphics editor and prototyping tool that can run as a desktop app or through the browser. The desktop app requires Windows 8.1 or later or macOS 10.12 or later. The browser requires Chrome 64+, Firefox 78+, Safari 13+ or Microsoft Edge 79+. Figma is free with limited features and requires a subscription ranging from $12 to $45 per month to access addition features.
